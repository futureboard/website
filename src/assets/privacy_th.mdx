# นโยบายความเป็นส่วนตัว Futureboard Marketplace

**อัปเดตล่าสุด:** 2026/06/03

นโยบายความเป็นส่วนตัวฉบับนี้อธิบายว่า **Futureboard Marketplace** (“Futureboard”, “เรา”, “ของเรา” หรือ “ผู้ให้บริการ”) เก็บรวบรวม ใช้ จัดเก็บ แบ่งปัน และปกป้องข้อมูลส่วนบุคคลอย่างไร เมื่อคุณเข้าถึงหรือใช้งาน Futureboard Marketplace รวมถึงเว็บไซต์ บริการ Marketplace เครื่องมือสำหรับ Creator ระบบบัญชีผู้ใช้ การเชื่อมต่อระบบชำระเงิน API การดาวน์โหลด และบริการอื่นที่เกี่ยวข้อง

เมื่อคุณใช้งาน Futureboard Marketplace ถือว่าคุณยอมรับการเก็บรวบรวมและการใช้ข้อมูลตามที่ระบุไว้ในนโยบายความเป็นส่วนตัวฉบับนี้ หากคุณไม่ยอมรับนโยบายนี้ โปรดอย่าใช้งาน Marketplace

---

## 1. ขอบเขตของนโยบายนี้

นโยบายความเป็นส่วนตัวฉบับนี้ใช้กับ Futureboard Marketplace และบริการที่เกี่ยวข้อง รวมถึง:

* การเรียกดูและค้นหาบน Marketplace
* บัญชีผู้ใช้และระบบยืนยันตัวตน
* โปรไฟล์ Creator และรายการคอนเทนต์
* การดาวน์โหลดคอนเทนต์ดิจิทัล
* การซื้อและขั้นตอนที่เกี่ยวข้องกับการชำระเงิน
* รีวิว คะแนน ความคิดเห็น และคำขอรับการสนับสนุน
* การตรวจสอบความปลอดภัย การสแกนมัลแวร์ และการป้องกันการใช้งานในทางที่ผิด
* Marketplace API และเครื่องมือสำหรับนักพัฒนา

นโยบายนี้ไม่ใช้กับเว็บไซต์ของบุคคลที่สาม ผู้ให้บริการชำระเงิน ผู้ให้บริการโฮสติ้ง ผู้ให้บริการวิเคราะห์ข้อมูล หรือบริการภายนอกที่ Futureboard ไม่ได้ควบคุม บริการเหล่านั้นอาจมีนโยบายความเป็นส่วนตัวของตนเอง

---

## 2. ข้อมูลที่เราเก็บรวบรวม

เราอาจเก็บรวบรวมข้อมูลหลายประเภท ขึ้นอยู่กับวิธีที่คุณใช้งาน Marketplace

### 2.1 ข้อมูลบัญชีผู้ใช้

เมื่อคุณสร้างบัญชีหรือเข้าสู่ระบบ เราอาจเก็บรวบรวม:

* ชื่อหรือชื่อที่แสดง
* อีเมล
* ชื่อผู้ใช้
* รูปโปรไฟล์หรืออวาตาร์
* รหัสผ่านหรือข้อมูลยืนยันตัวตน ในกรณีที่เกี่ยวข้อง
* ตัวระบุจาก OAuth หรือการเข้าสู่ระบบผ่านบริการภายนอก
* การตั้งค่าบัญชีและค่ากำหนดต่าง ๆ
* ข้อมูลโปรไฟล์ Creator

### 2.2 ข้อมูล Creator

หากคุณลงทะเบียนเป็น Creator หรือเผยแพร่คอนเทนต์ เราอาจเก็บรวบรวม:

* ชื่อ Creator หรือชื่อแบรนด์
* ข้อมูลโปรไฟล์สาธารณะ
* รายการคอนเทนต์และเมตาดาตา
* ไฟล์ที่อัปโหลดและทรัพยากรที่เกี่ยวข้อง
* ข้อมูลใบอนุญาต
* ข้อมูลติดต่อสำหรับการสนับสนุน
* สถานะการเชื่อมต่อบัญชีรับชำระเงิน
* เมตาดาตาที่เกี่ยวข้องกับภาษีหรือการรับเงิน ในกรณีที่จำเป็น
* ข้อมูลประสิทธิภาพบน Marketplace เช่น ยอดขาย ยอดดาวน์โหลด การคืนเงิน และคะแนนรีวิว

ข้อมูลบางส่วนของ Creator อาจแสดงต่อสาธารณะบน Marketplace

### 2.3 กิจกรรมบน Marketplace

เมื่อคุณใช้งาน Marketplace เราอาจเก็บรวบรวม:

* หน้าที่เข้าชม
* คำค้นหา
* การเข้าชมหน้าคอนเทนต์
* การดาวน์โหลด
* การซื้อ
* รีวิวและคะแนน
* รายการโปรดหรือรายการที่บันทึกไว้
* การรายงานคอนเทนต์
* คำขอรับการสนับสนุน
* กิจกรรมของอุปกรณ์และเซสชัน
* แหล่งที่มาของการเข้าชม

### 2.4 ข้อมูลการชำระเงิน

การชำระเงินอาจดำเนินการโดยผู้ให้บริการชำระเงินภายนอก เช่น Stripe หรือผู้ให้บริการอื่นที่รองรับ

Futureboard จะไม่จัดเก็บหมายเลขบัตรชำระเงินแบบเต็ม เว้นแต่จะระบุไว้อย่างชัดเจนและสามารถทำได้ตามกฎหมาย อย่างไรก็ตาม เราอาจได้รับและจัดเก็บข้อมูลที่เกี่ยวข้องกับการชำระเงินในขอบเขตจำกัด เช่น:

* หมายเลขธุรกรรม
* ประวัติการซื้อ
* สถานะการชำระเงิน
* สถานะการคืนเงิน
* สถานะการปฏิเสธการชำระเงินหรือข้อพิพาท
* รหัสลูกค้าจากผู้ให้บริการชำระเงิน
* สถานะการจ่ายเงินให้ Creator
* ประเทศหรือภูมิภาคสำหรับการเรียกเก็บเงิน
* เมตาดาตาที่เกี่ยวข้องกับภาษี ในกรณีที่จำเป็น

ผู้ให้บริการชำระเงินอาจเก็บรวบรวมและประมวลผลข้อมูลการชำระเงินของคุณภายใต้ข้อกำหนดและนโยบายความเป็นส่วนตัวของผู้ให้บริการนั้น

### 2.5 คอนเทนต์และไฟล์ที่อัปโหลด

เมื่อ Creator อัปโหลดคอนเทนต์ เราอาจเก็บรวบรวมและประมวลผล:

* ไฟล์ที่อัปโหลด
* ชื่อไฟล์
* ขนาดไฟล์
* ประเภทไฟล์
* ข้อมูลเวอร์ชัน
* ค่า checksum หรือ hash
* เมตาดาตา
* ภาพตัวอย่างและไฟล์พรีวิว
* ไฟล์เอกสารประกอบ
* ไฟล์ใบอนุญาต
* ผลการตรวจสอบความปลอดภัย

เราอาจประมวลผลไฟล์ที่อัปโหลดเพื่อการโฮสต์ การส่งมอบคอนเทนต์ การกลั่นกรอง การสแกนมัลแวร์ การตรวจสอบความเข้ากันได้ การจัดทำดัชนี และการดำเนินงานของ Marketplace

### 2.6 ข้อมูลอุปกรณ์ บันทึกระบบ และข้อมูลทางเทคนิค

เราอาจเก็บรวบรวมข้อมูลทางเทคนิคโดยอัตโนมัติ รวมถึง:

* ที่อยู่ IP
* ประเภทและเวอร์ชันของเบราว์เซอร์
* ระบบปฏิบัติการ
* ประเภทอุปกรณ์
* การตั้งค่าภาษา
* เขตเวลา
* URL อ้างอิง
* หน้าที่เข้าชม
* คำขอ API
* บันทึกการแครช
* บันทึกข้อผิดพลาด
* บันทึกการดาวน์โหลด
* บันทึกด้านความปลอดภัย
* ตำแหน่งโดยประมาณที่ได้จากที่อยู่ IP

ข้อมูลนี้ช่วยให้เราดำเนินงาน รักษาความปลอดภัย แก้ไขข้อผิดพลาด และปรับปรุง Marketplace

### 2.7 การติดต่อสื่อสาร

หากคุณติดต่อเรา รายงานบั๊ก ส่งคำขอรับการสนับสนุน ยื่นคำร้องเรียนด้านทรัพย์สินทางปัญญา หรือติดต่อกับ Creator ผ่าน Marketplace เราอาจเก็บรวบรวม:

* เนื้อหาข้อความของคุณ
* ข้อมูลติดต่อ
* ไฟล์แนบ
* ประวัติการสนับสนุน
* ข้อมูลบัญชีและธุรกรรมที่เกี่ยวข้อง

## 3. วิธีที่เราใช้ข้อมูล

เราใช้ข้อมูลที่เก็บรวบรวมเพื่อ:

* ให้บริการ ดำเนินงาน และดูแล Marketplace
* สร้างและจัดการบัญชีผู้ใช้
* ยืนยันตัวตนผู้ใช้และปกป้องบัญชี
* ให้ Creator สามารถอัปโหลด เผยแพร่ จัดการ และขายคอนเทนต์
* ให้ลูกค้าสามารถค้นหา ซื้อ ดาวน์โหลด และรีวิวคอนเทนต์
* ประมวลผลการซื้อ การคืนเงิน ข้อพิพาท และบันทึกธุรกรรม
* เชื่อมต่อ Creator กับผู้ให้บริการชำระเงิน
* แสดงโปรไฟล์ Creator และรายการคอนเทนต์แบบสาธารณะ
* ส่งมอบคอนเทนต์ดิจิทัลและจัดการสิทธิ์การดาวน์โหลด
* ให้การสนับสนุนลูกค้าและตอบคำถาม
* ตรวจจับ ป้องกัน และสอบสวนการฉ้อโกง การใช้งานในทางที่ผิด มัลแวร์ สแปม และภัยคุกคามด้านความปลอดภัย
* กลั่นกรองคอนเทนต์ รีวิว และกิจกรรมของผู้ใช้
* บังคับใช้ข้อกำหนดการให้บริการและนโยบายของ Marketplace
* ปรับปรุงประสิทธิภาพ ความน่าเชื่อถือ การค้นหา คำแนะนำ และประสบการณ์ผู้ใช้
* วิเคราะห์การใช้งาน Marketplace และประสิทธิภาพทางธุรกิจ
* ปฏิบัติตามภาระหน้าที่ทางกฎหมาย ภาษี บัญชี และกฎระเบียบ

## 4. ฐานทางกฎหมายในการประมวลผลข้อมูล

ขึ้นอยู่กับที่ตั้งของคุณ เราอาจประมวลผลข้อมูลส่วนบุคคลภายใต้ฐานทางกฎหมายอย่างใดอย่างหนึ่งหรือหลายอย่าง รวมถึง:

* **ความจำเป็นตามสัญญา:** เพื่อให้บริการ Marketplace ที่คุณร้องขอ
* **ความยินยอม:** ในกรณีที่คุณให้ความยินยอม เช่น การสื่อสารทางเลือก หรือคุกกี้ในกรณีที่กฎหมายกำหนด
* **ประโยชน์โดยชอบด้วยกฎหมาย:** เพื่อดำเนินงาน รักษาความปลอดภัย ปรับปรุง และปกป้อง Marketplace
* **ภาระหน้าที่ตามกฎหมาย:** เพื่อปฏิบัติตามข้อกำหนดด้านภาษี บัญชี การป้องกันการฉ้อโกง การบังคับใช้กฎหมาย หรือกฎระเบียบ
* **ประโยชน์สำคัญต่อชีวิตหรือประโยชน์สาธารณะ:** ในกรณีที่กฎหมายที่เกี่ยวข้องกำหนด

## 5. ข้อมูลที่เปิดเผยต่อสาธารณะ

ข้อมูลบางประเภทอาจมองเห็นได้ต่อสาธารณะบน Marketplace รวมถึง:

* ชื่อที่แสดงของ Creator
* โปรไฟล์ Creator
* รายการคอนเทนต์สาธารณะ
* คำอธิบายคอนเทนต์
* ภาพหน้าจอและสื่อพรีวิว
* ข้อมูลใบอนุญาต
* รีวิวและคะแนนสาธารณะ
* ความคิดเห็นหรือการตอบกลับสาธารณะ
* สถิติการดาวน์โหลดหรือยอดขาย ในกรณีที่มีการแสดงผล

โปรดอย่าเผยแพร่ข้อมูลส่วนบุคคลที่คุณไม่ต้องการให้ปรากฏต่อสาธารณะ

## 6. การแบ่งปันข้อมูล

เราอาจแบ่งปันข้อมูลในสถานการณ์ต่อไปนี้:

### 6.1 กับผู้ให้บริการ

เราอาจแบ่งปันข้อมูลกับผู้ให้บริการที่เชื่อถือได้ ซึ่งช่วยให้เราดำเนินงาน Marketplace เช่น:

* ผู้ให้บริการโฮสติ้ง
* ผู้ให้บริการฐานข้อมูล
* ผู้ให้บริการพื้นที่จัดเก็บข้อมูลและ CDN
* ผู้ประมวลผลการชำระเงิน
* ผู้ให้บริการยืนยันตัวตน
* ผู้ให้บริการส่งอีเมล
* ผู้ให้บริการวิเคราะห์ข้อมูล
* ผู้ให้บริการสแกนความปลอดภัยและมัลแวร์
* เครื่องมือสนับสนุนลูกค้า
* บริการบันทึกระบบและติดตามสถานะ

ผู้ให้บริการเหล่านี้อาจประมวลผลข้อมูลเท่าที่จำเป็นเพื่อให้บริการแก่เรา

### 6.2 กับผู้ให้บริการชำระเงิน

เมื่อคุณทำการซื้อหรือเชื่อมต่อบัญชีรับชำระเงินของ Creator ข้อมูลที่เกี่ยวข้องอาจถูกแบ่งปันกับผู้ให้บริการชำระเงิน เพื่อประมวลผลการชำระเงิน การจ่ายเงิน การคืนเงิน การปฏิเสธการชำระเงิน ภาษี การป้องกันการฉ้อโกง และการตรวจสอบการปฏิบัติตามข้อกำหนด

### 6.3 ระหว่างลูกค้าและ Creator

เราอาจแบ่งปันข้อมูลในขอบเขตจำกัดระหว่างลูกค้าและ Creator เมื่อจำเป็นสำหรับ:

* การส่งมอบการซื้อ
* การตรวจสอบใบอนุญาต
* การสนับสนุนลูกค้า
* การจัดการคืนเงินหรือข้อพิพาท
* การอัปเดตคอนเทนต์
* การสอบสวนการใช้งานในทางที่ผิด

เราไม่มีเจตนาที่จะแบ่งปันข้อมูลบัตรชำระเงินแบบเต็มให้กับ Creator

### 6.4 ด้วยเหตุผลทางกฎหมายและความปลอดภัย

เราอาจเปิดเผยข้อมูล หากเราเชื่อว่าจำเป็นเพื่อ:

* ปฏิบัติตามกฎหมายหรือกระบวนการทางกฎหมายที่เกี่ยวข้อง
* ตอบสนองต่อคำขอที่ชอบด้วยกฎหมายจากหน่วยงานรัฐ
* บังคับใช้ข้อกำหนดการให้บริการ
* สอบสวนการฉ้อโกง การใช้งานในทางที่ผิด มัลแวร์ หรือภัยคุกคามด้านความปลอดภัย
* ปกป้องสิทธิ ความปลอดภัย ทรัพย์สิน หรือความมั่นคงของ Futureboard ผู้ใช้ Creator ลูกค้า หรือสาธารณะ

### 6.5 การโอนธุรกิจ

หาก Futureboard หรือ Marketplace มีส่วนเกี่ยวข้องกับการควบรวมกิจการ การซื้อกิจการ การปรับโครงสร้าง การขายทรัพย์สิน การจัดหาเงินทุน หรือธุรกรรมในลักษณะใกล้เคียงกัน ข้อมูลอาจถูกโอนเป็นส่วนหนึ่งของธุรกรรมนั้น

---

## 7. คุกกี้และเทคโนโลยีที่คล้ายกัน

เราอาจใช้คุกกี้ local storage, pixels, SDKs หรือเทคโนโลยีที่คล้ายกัน เพื่อ:

* ให้คุณอยู่ในระบบ
* จดจำค่ากำหนด
* รักษาความปลอดภัยของบัญชี
* ป้องกันการฉ้อโกงและการใช้งานในทางที่ผิด
* วิเคราะห์การใช้งาน
* ปรับปรุงประสิทธิภาพ
* รองรับฟีเจอร์ของ Marketplace
* วัดผลกิจกรรมด้านการตลาดหรือแหล่งอ้างอิง

คุณอาจควบคุมคุกกี้ได้ผ่านการตั้งค่าเบราว์เซอร์ การปิดใช้งานคุกกี้อาจส่งผลต่อการทำงานบางส่วนของ Marketplace

ในกรณีที่กฎหมายกำหนด เราอาจขอความยินยอมจากคุณก่อนใช้คุกกี้หรือเทคโนโลยีติดตามบางประเภท

---

## 8. ข้อมูลวิเคราะห์และประสิทธิภาพ

เราอาจใช้เครื่องมือวิเคราะห์ข้อมูลเพื่อทำความเข้าใจว่าผู้ใช้มีปฏิสัมพันธ์กับ Marketplace อย่างไร ซึ่งอาจรวมถึงข้อมูล เช่น หน้าที่เข้าชม คำค้นหา การใช้งานฟีเจอร์ ค่าประสิทธิภาพ เหตุการณ์แครช และตำแหน่งโดยประมาณ

เราใช้ข้อมูลนี้เพื่อปรับปรุงความน่าเชื่อถือ การใช้งานง่าย การค้นพบคอนเทนต์ ประสิทธิภาพ และความปลอดภัย

---

## 9. ความปลอดภัย

เราใช้มาตรการทางเทคนิคและองค์กรอย่างสมเหตุสมผลเพื่อปกป้องข้อมูลส่วนบุคคล รวมถึงการควบคุมการเข้าถึง การเข้ารหัสในกรณีที่เหมาะสม การติดตาม การบันทึกระบบ การตรวจสอบความปลอดภัย และการตรวจจับการใช้งานในทางที่ผิด

อย่างไรก็ตาม ไม่มีบริการออนไลน์ใดที่สามารถรับประกันความปลอดภัยได้อย่างสมบูรณ์ คุณมีหน้าที่ปกป้องข้อมูลเข้าสู่ระบบของบัญชี และใช้รหัสผ่านที่แข็งแรงหรือวิธีการยืนยันตัวตนที่ปลอดภัย

หากคุณเชื่อว่าบัญชีหรือข้อมูลของคุณถูกเข้าถึงโดยไม่ได้รับอนุญาต โปรดติดต่อเราทันที

---

## 10. ระยะเวลาการเก็บรักษาข้อมูล

เราเก็บรักษาข้อมูลส่วนบุคคลเท่าที่จำเป็นอย่างสมเหตุสมผลสำหรับวัตถุประสงค์ที่ระบุในนโยบายความเป็นส่วนตัวฉบับนี้ รวมถึง:

* การให้บริการ Marketplace
* การดูแลบัญชีผู้ใช้
* การประมวลผลธุรกรรม
* การดูแลบันทึกของ Creator และลูกค้า
* การปฏิบัติตามภาระหน้าที่ทางกฎหมาย ภาษี บัญชี และการรายงาน
* การแก้ไขข้อพิพาท
* การป้องกันการฉ้อโกง การใช้งานในทางที่ผิด และภัยคุกคามด้านความปลอดภัย
* การบังคับใช้ข้อกำหนดการให้บริการ

ข้อมูลบางประเภทอาจถูกเก็บรักษาหลังการลบบัญชี ในกรณีที่กฎหมายกำหนด จำเป็นต่อวัตถุประสงค์ทางธุรกิจโดยชอบด้วยกฎหมาย หรือจำเป็นต่อความปลอดภัย การป้องกันการฉ้อโกง การบัญชี หรือการแก้ไขข้อพิพาท

---

## 11. สิทธิและทางเลือกของคุณ

ขึ้นอยู่กับที่ตั้งของคุณ คุณอาจมีสิทธิเกี่ยวกับข้อมูลส่วนบุคคลของคุณ รวมถึงสิทธิในการ:

* เข้าถึงข้อมูลส่วนบุคคลที่เรามีเกี่ยวกับคุณ
* แก้ไขข้อมูลที่ไม่ถูกต้องหรือไม่สมบูรณ์
* ลบข้อมูลส่วนบุคคลของคุณ
* จำกัดหรือคัดค้านการประมวลผลบางประเภท
* ถอนความยินยอม ในกรณีที่การประมวลผลอิงจากความยินยอม
* ขอสำเนาข้อมูลของคุณ
* ขอให้โอนย้ายข้อมูล
* ยกเลิกการรับการสื่อสารบางประเภท
* ยื่นเรื่องร้องเรียนต่อหน่วยงานคุ้มครองข้อมูลส่วนบุคคล

หากต้องการใช้สิทธิของคุณ โปรดติดต่อเราตามช่องทางด้านล่าง เราอาจต้องยืนยันตัวตนของคุณก่อนดำเนินการตามคำขอบางประเภท

---

## 12. การลบบัญชี

คุณสามารถขอลบบัญชีของคุณได้

เมื่อบัญชีของคุณถูกลบ เราอาจลบหรือทำให้ข้อมูลส่วนบุคคลบางส่วนไม่สามารถระบุตัวตนได้ อย่างไรก็ตาม ข้อมูลบางอย่างอาจยังคงถูกเก็บรักษาไว้เมื่อจำเป็น เช่น:

* บันทึกธุรกรรม
* บันทึกด้านภาษีหรือบัญชี
* บันทึกด้านความปลอดภัย
* บันทึกเพื่อป้องกันการใช้งานในทางที่ผิด
* บันทึกข้อพิพาททางกฎหมาย
* คอนเทนต์สาธารณะที่คุณเลือกเผยแพร่
* ข้อมูลสำรองในระยะเวลาจำกัด

หากคุณเป็น Creator การลบบัญชีอาจส่งผลต่อการเข้าถึงเครื่องมือ Creator รายการคอนเทนต์ การรับเงิน ภาระหน้าที่ในการสนับสนุน และการจัดการคอนเทนต์

---

## 13. การสื่อสารทางการตลาด

เราอาจส่งข้อความที่เกี่ยวข้องกับบริการให้คุณ เช่น การแจ้งเตือนบัญชี ใบเสร็จธุรกรรม การแจ้งเตือนด้านความปลอดภัย การอัปเดตนโยบาย และข้อความสนับสนุน

ในกรณีที่กฎหมายกำหนดและคุณให้ความยินยอม เราอาจส่งข่าวสารผลิตภัณฑ์ ข่าว Marketplace ประกาศจาก Creator หรือข้อความส่งเสริมการขาย

คุณสามารถยกเลิกการรับข้อความทางการตลาดได้ทุกเมื่อ อย่างไรก็ตาม ข้อความที่จำเป็นต่อการให้บริการอาจยังคงถูกส่งเมื่อจำเป็น

---

## 14. ความเป็นส่วนตัวของเด็ก

Futureboard Marketplace ไม่ได้มีไว้สำหรับเด็กที่มีอายุต่ำกว่าเกณฑ์ที่กฎหมายที่เกี่ยวข้องกำหนด

เราไม่ได้ตั้งใจเก็บรวบรวมข้อมูลส่วนบุคคลจากเด็กโดยไม่มีความยินยอมที่เหมาะสม หากคุณเชื่อว่าเด็กได้ให้ข้อมูลส่วนบุคคลแก่เราโดยไม่มีความยินยอมที่ถูกต้อง โปรดติดต่อเราเพื่อให้เราดำเนินการตามสมควร

---

## 15. การโอนข้อมูลระหว่างประเทศ

Futureboard Marketplace อาจดำเนินงานโดยใช้ผู้ให้บริการที่อยู่ในหลายประเทศ ข้อมูลของคุณอาจถูกโอนไป จัดเก็บ หรือประมวลผลในประเทศอื่นนอกเหนือจากประเทศของคุณ

ในกรณีที่กฎหมายกำหนด เราจะใช้มาตรการคุ้มครองที่เหมาะสมสำหรับการโอนข้อมูลระหว่างประเทศ

---

## 16. ลิงก์และบริการของบุคคลที่สาม

Marketplace อาจมีลิงก์ไปยังเว็บไซต์ของบุคคลที่สาม เว็บไซต์ของ Creator เอกสารประกอบ หน้าโซเชียลมีเดีย บริการชำระเงิน หรือเครื่องมือภายนอก

เราไม่รับผิดชอบต่อแนวปฏิบัติด้านความเป็นส่วนตัว เนื้อหา ความปลอดภัย หรือนโยบายของบริการบุคคลที่สาม โปรดตรวจสอบนโยบายความเป็นส่วนตัวของบริการเหล่านั้นก่อนให้ข้อมูลแก่บริการนั้น

---

## 17. โอเพนซอร์สและการมีส่วนร่วมของชุมชน

Futureboard อาจดำเนินโครงการโอเพนซอร์ส พื้นที่ชุมชน ระบบติดตามปัญหา repository ฟอรัม หรือระบบเอกสาร

ข้อมูลที่คุณโพสต์แบบสาธารณะในพื้นที่ชุมชนอาจมองเห็นได้โดยผู้อื่น โปรดหลีกเลี่ยงการโพสต์ข้อมูลส่วนตัวหรือข้อมูลอ่อนไหวในช่องทางสาธารณะ

---

## 18. การเปลี่ยนแปลงนโยบายความเป็นส่วนตัวนี้

เราอาจอัปเดตนโยบายความเป็นส่วนตัวฉบับนี้เป็นครั้งคราว

เมื่อมีการเปลี่ยนแปลงที่มีนัยสำคัญ เราอาจแจ้งให้คุณทราบผ่าน Marketplace เว็บไซต์ อีเมล หรือช่องทางอื่นที่เหมาะสม นโยบายความเป็นส่วนตัวฉบับอัปเดตจะมีผลเมื่อเผยแพร่ เว้นแต่จะระบุไว้เป็นอย่างอื่น

การใช้งาน Marketplace ต่อไปหลังจากนโยบายความเป็นส่วนตัวฉบับอัปเดตมีผล ถือว่าคุณรับทราบนโยบายฉบับอัปเดตแล้ว
